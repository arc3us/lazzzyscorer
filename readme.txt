# lazzzyscorer
A selenium based script to hack/automate games on zzzscore

This is a python script which automate games by zzzscore (http://zzzscore.com) 


Currently this works on three games:
  1- 1to50 - finishes in 2.45 seconds 
  2- Color blind test - typically scores 191 
  3- Color blind test 2 - typically scores 191

Prerequisites:
  Python
  Selenium
  Google Chrome
  Chromedriver for Selenium

Usage: 
  Run the desired script to automate the game.
